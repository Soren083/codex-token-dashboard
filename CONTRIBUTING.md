# Contributing

Thanks for helping improve Codex Token Dashboard.

## Development

```bash
npm test
node --check server.mjs
```

The app intentionally has no runtime dependencies. Keep changes small and prefer plain Node.js and browser APIs unless a dependency clearly reduces maintenance risk.

## Pull requests

- Describe the user-facing change.
- Include the checks you ran.
- Avoid committing local Codex session files, logs, generated launchd plists, or secrets.
